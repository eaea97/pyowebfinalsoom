명상과 마음챙김을 위한 종합 웹사이트입니다.

## 주요 기능

- 명상 센터 위치 안내
- 명상 비디오 콘텐츠
- 블로그 및 커뮤니티
- 명상 전문가 소개
- 이벤트 일정

## 기술 스택
